# SVSA

A community/club management application made for managing finances of the community.

## Technologies used

- NextJS
- TypeScript
- tRPC
- AuthJS
- Prisma
- Tailwind CSS
- Zod
- Yup
- React Hook Forms
- PostgreSQL
- UI Primitives by [Shadcn](https://ui.shadcn.com/)

## Hosting

- Hosted on Vercel
- Database on Supabase
- Logs on Axiom
